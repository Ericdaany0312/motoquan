# 摩托圈运维操作手册

## 克隆项目（首次部署）

```bash
cd ~
git clone https://github.com/Ericdaany0312/motoquan.git
cd motoquan
```

## 配置环境变量

```bash
cat > ~/motoquan/.env.local << 'EOF'
NEXT_PUBLIC_ADMIN_EMAIL=admin@motoquan.com
NEXT_PUBLIC_ADMIN_PASSWORD=motoquan2026
EOF
```

## 安装依赖

```bash
cd ~/motoquan
npm install
```

## 构建

```bash
npm run build
```

## 启动服务（前台）

```bash
npm start -p 3000
```

## 启动服务（后台 screen）

```bash
screen -S motoquan
npm start -p 3000
# 按 Ctrl+A D 退出会话
```

## 重启服务

```bash
screen -r motoquan
# Ctrl+C 停止
npm start -p 3000
# Ctrl+A D 退出
```

## 查看服务状态

```bash
screen -r motoquan
```

## 更新代码

```bash
cd ~/motoquan
git pull origin main
npm run build
# 重启服务
```

## 常用端口

- 3000：主服务端口
- 22：SSH

## 防火墙

```bash
sudo ufw status
```

## 查看日志（journalctl）

```bash
sudo journalctl -u sshd -n 20 --no-pager
```

## 目录结构

- `~/motoquan`：项目根目录
- `~/motoquan/.env.local`：环境变量（勿泄露）
- `~/motoquan/src`：源代码
